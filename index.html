About me

Hi, I’m Diogo Ribeiro, a Brazilian software engineer. Coding has been my hobby since I was 17 when I used to build simple web pages using HTML and CSS just for fun. Once I got in touch with software development I knew that was what I wanted to work on because it amazed me, I loved and still love the fact that I can write computer instructions that are going to give life to computers.

I’ve been working professionally as a software engineer since 2010 ever since I’ve had the opportunity to work with a lot of different technologies (javascript, ruby, .net, java, groovy), platforms (web, desktop, embedded), frameworks (Rails, React, Grails), always focusing on delivering high-quality software, well tested, with beautiful design, with value for the client and in a fair amount of time.

I've been working remotely for 2 years now, moreover, I'm also helping mentoring young devs which helped me to improve my communication skills
