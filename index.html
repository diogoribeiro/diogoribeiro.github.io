About me

Hi, I’m Diogo Ribeiro, a Brazilian software engineer. Coding has been my hobby since I was 17 when I used to build simple web pages using HTML and CSS just for fun.

I’ve been working professionally as a software engineer since 2010 and since then I’ve had the opportunity to work with lots of different types of software like personal websites, e-commerce, CMS, cryptocurrency bots, etc., always focusing on delivering high-quality software, well tested, with value for the client and in a fair amount of time.

Front end

Throughout of the years, I’ve been working as a software engineer I’ve had the opportunity to work on projects that used interesting front-end technologies such as react, sass, handlebars, jQuery, HTML5, etc.

Back end

I also worked on the back end side of projects using Ruby and the popular framework Ruby on Rails, Java and its main frameworks like spring and hibernate, Node.js and Express.

Lately, I’ve been working remote building software for startups and also fortune 500 companies.

In this time working remote I got the opportunity to mentor young devs and lead a team, challenges that changed the way I communicate and plan tasks.
